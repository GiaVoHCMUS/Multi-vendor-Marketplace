Pagination:
  1. Offset Pagination
    {
      "success": true,
      "message": "..."
      "data": [...],
      "meta": {
        "currentPage": 2,
        "limit": 10,
        "totalItems": 120,
        "totalPages": 12
      }
    }
  
  2. Cursor Pagination
    {
      "success": true,
      "message": "..."
      "data": [...],
      "meta": {
        "limit": 10,
        "hasMore": true,
        "nextCursor": "ckf93jd"
      }
    } 

    next_cursor (string/opaque token): Giá trị con trỏ của bản ghi cuối cùng trong trang hiện tại, dùng để truyền vào tham số after cho yêu cầu tiếp theo.
    previous_cursor (string/opaque token): Giá trị con trỏ của bản ghi đầu tiên, dùng để quay lại trang trước (before).
    has_more (boolean): Cho biết còn dữ liệu phía sau hay không.
    limit (integer): Số lượng item thực tế đã trả về trong trang này.
